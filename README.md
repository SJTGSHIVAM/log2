Log2 is a book store where you can find books from different domains of computer science. Right now only UI is implemented.

## Features

A few features you get with Log2:

* Responsive design
* Dark/Light theme
* Home page
* Product listing page
* Filters 
* Cart management
* Wishlist management
* Sign-up
* Address Management
* Login
* Logout
* Forget Password
* Search
